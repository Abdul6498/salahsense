# Phase 1 Plan

## Objective

Build a proof-of-concept vertical state machine to count rakats.

## Included in Phase 1

- Standing and sujud Y-axis threshold mapping
- Movement direction detection (`GOING_DOWN`, `GOING_UP`)
- Sequence validation for one rakat pattern
- Local output to terminal

## Deferred

- Full ruku detection
- Fine-grained hand/finger checks
- Cloud dashboard integration
